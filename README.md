# Case Study

## Getting Started

First, install all necessary items to build and run the project.

```bash
npm install
# or
yarn install
```

## Tests

### Unit Tests (Jest)

```bash
npm run test:jest
# or
yarn test:jest
```

## Environments

### Development

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Poduction

#### Pre-production

```bash
npm run build
# or
yarn build
```

#### Post-production

```bash
npm run start
# or
yarn start
```

## Assumptions

- Material UI library would allow for more themed items.
